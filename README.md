# 修真炼器录游戏说明

## 游戏介绍
这是一个修真主题的挖矿炼器游戏，玩家可以通过挖矿获得材料，然后熔炼和锻造各种装备。

## 如何运行游戏

### 方式一：直接打开HTML文件（推荐）
1. 在文件资源管理器中找到 `deepseek_html_20250728_0c94c1.html` 文件
2. 双击该文件，使用默认浏览器打开
3. 开始游戏

### 方式二：使用浏览器打开
1. 打开任意现代浏览器（Chrome、Firefox、Edge等）
2. 点击浏览器菜单中的「打开文件」或「打开」选项
3. 选择 `deepseek_html_20250728_0c94c1.html` 文件
4. 开始游戏

## 游戏功能

### 1. 挖矿系统
- 选择不同等级的矿脉进行挖矿
- 消耗体力获得矿石和其他资源
- 提升挖矿技能等级以开采高级矿脉

### 2. 炼器系统
- **熔炼**：将矿石熔炼成金属锭
- **锻造**：使用金属锭和其他材料锻造武器、防具等装备
- 锻造结果有不同品质（普通、精良、稀有、史诗、传说）

### 3. 拍卖行系统
- 浏览其他玩家上架的物品
- 将自己的物品上架出售
- 购买所需的材料和装备

### 4. 任务系统
- 接受并完成各种任务
- 获得金币、经验和其他奖励

### 5. 背包和装备
- 管理自己的物品和装备
- 装备武器和防具提升属性

## 游戏操作

1. **切换界面**：点击左侧导航栏的按钮切换不同功能界面
2. **挖矿**：在挖矿界面选择矿脉，点击「挖矿」按钮
3. **熔炼**：在炼器界面选择材料和数量，点击「熔炼」按钮
4. **锻造**：在炼器界面选择要锻造的物品，点击「锻造」按钮
5. **交易**：在拍卖行界面浏览物品，点击「购买」按钮购买物品
6. **任务**：在任务界面接受和提交任务

## 游戏数据

游戏数据存储在浏览器本地，刷新页面后数据不会丢失。
